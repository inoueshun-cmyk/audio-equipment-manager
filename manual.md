## 1. ログイン・ログアウト

### ログイン手順
1. URLにアクセスするとログイン画面が表示される
2. **ロール**を選択する（管理者 / 一般メンバー）
3. **パスワード**を入力して「ログイン」をタップ

| ロール | 初期パスワード |
|--------|--------------|
| 管理者 | admin123 |
| 一般メンバー | 0000 |

> **注意：** 初回ログイン後、必ずパスワードを変更してください（設定画面から変更可能）。

### ログアウト
画面右上の「ログアウト」ボタンをタップ。

---

## 2. ダッシュボード

ログイン後に最初に表示される画面。以下の情報を確認できる。

- **総機材種類** — 登録されている機材の種類数
- **資産総額** — 全機材の（数量 × 単価）の合計
- **要対応** — 状態が「要修理」または「故障」の機材数
- **今後のイベント** — 本日以降に予定されているイベント数
- **点検アラート** — 次回点検日が過ぎている機材数（オレンジ色で警告表示）

### 点検アラート
期限が過ぎている機材は**黄色いアラートボックス**に一覧表示される。速やかに点検・記録を行うこと。

## 3. 機材マスタ

音響機材の在庫情報を管理する画面。

### 機材の検索・絞り込み
- **検索ボックス** — 機材名・メーカー・型番で絞り込み
- **カテゴリ選択** — 入力系 / 処理系 / 出力系 / 周辺 で絞り込み

### 機材の登録（管理者のみ）
1. 「**+ 追加**」ボタンをタップ
2. 以下の項目を入力して「保存」

| 項目 | 説明 |
|------|------|
| 機材名 | 例：SM58、パワーアンプ |
| メーカー | 例：Shure、Yamaha |
| 型番 | 例：SM58-LC、MG16XU |
| カテゴリ | 入力系 / 処理系 / 出力系 / 周辺 |
| 状態 | 良好 / 要注意 / 要修理 / 故障 |
| 数量 | 所有台数 |
| 単価（日） | 見積書に使用するレンタル単価 |
| 保管場所 | 例：棚A-1 |
| シリアル番号 | 任意 |
| 最終点検日 | 直近の点検日 |
| 次回点検予定 | 次回点検日（アラート基準日） |
| メモ | 備考など |

### 機材の編集・削除（管理者のみ）
- 各行の「**編集**」ボタンから内容を変更できる
- 「**削除**」ボタンで削除（削除は取り消しできないため注意）

## 4. イベント管理

現場・イベントごとの機材リストと費用項目を管理する画面。

### イベントの作成（管理者のみ）
1. 「**+ 追加**」ボタンをタップ
2. 以下を入力して「保存」

| 項目 | 説明 |
|------|------|
| イベント名 | 例：春のライブイベント |
| クライアント名 | 例：〇〇音楽祭実行委員会 |
| 開始日・終了日 | 現場の日程 |
| ステータス | 準備中 / 実施済み / キャンセル |
| 備考 | 任意 |

### 機材リストの追加（管理者のみ）
1. 左の一覧からイベントを選択
2. 「機材リスト」セクションで機材・数量・日数を選んで「**追加**」
3. 追加された機材の小計が自動計算される

### 費用項目の追加（管理者のみ）
1. 「費用項目」セクションのプリセットチップをタップして追加
   - 例：オペレーター、搬入作業、搬出作業、車両費 など
2. 「**+ カスタム**」で任意の項目も追加可能
3. 追加後は**項目名・数量・日数・単価**をその場で直接編集できる

> 機材小計＋費用小計の合計がリアルタイムで表示される。

### 見積書への連携
イベント詳細右上の「**見積書 ↗**」ボタンをタップすると、そのイベントの内容が見積書タブに自動反映される。

## 5. 見積書

イベントの内容をもとに見積書を作成・印刷する画面。

### 見積書の作成
1. 「**イベント選択**」でイベントを選ぶ（機材・費用が自動反映）
2. 必要に応じて以下を編集（管理者のみ）

| 項目 | 説明 |
|------|------|
| 発行会社名 | 自社名 |
| 宛先 | クライアント名 |
| 発行日 | 見積書の日付 |
| 消費税率% | デフォルト10% |
| 備考 | 印刷時に記載される注意事項など |

3. 金額は**自動計算**される（小計・消費税・合計）

### 印刷・PDF保存
「**印刷**」ボタンをタップ → ブラウザの印刷ダイアログが開く。  
「PDFとして保存」を選択するとPDFファイルとして保存できる。

> 一般メンバーは見積書の**閲覧・印刷のみ**可能。内容の編集は管理者のみ。

---

## 6. メンテナンス

機材の点検記録を管理する画面。管理者・一般メンバーともに記録の追加が可能。

### 点検アラートの確認
次回点検日が過ぎている機材が画面上部に一覧表示される。

### 点検記録の追加
1. 「**機材**」で対象機材を選択
2. 「**点検日**」を入力（デフォルトは本日）
3. 「**状態**」を選択（良好 / 要注意 / 要修理 / 故障）
4. 必要に応じて「**メモ**」を入力
5. 「**記録**」ボタンをタップ

> 点検記録を追加すると、機材マスタの状態と最終点検日が自動更新される。

### 点検履歴の確認
画面下部に新しい順で点検履歴が表示される。

## 7. 設定（管理者のみ）

### 費用項目プリセット管理
イベント管理画面の費用項目に表示されるプリセットを編集できる。

1. 既存の項目名を直接編集可能
2. 「**削除**」ボタンで不要な項目を削除
3. 下部の入力欄から新しい項目を「**追加**」
4. 「**プリセットを保存**」ボタンで変更を確定

> 保存するまで変更は反映されないので注意。

### パスワード管理

**管理者パスワードの変更**
1. 「現在のパスワード」を入力
2. 「新しいパスワード」を入力
3. 「管理者パスワードを変更」をタップ

**一般メンバーパスワードの変更**
1. 「新しいパスワード」を入力
2. 「変更」をタップ

> パスワードはFirebaseに保存される。忘れた場合はFirebaseコンソールから直接確認・変更が可能。
